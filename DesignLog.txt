==Power Module and Serial Module
Power and Serial Module:
104 Capacitor x4 for decoupling
100uF Capacitor x1 for Power Source
LED x5 for Power Signal,TX1,RX1,TX2 and RX2.
3A Polyfuse x1 for Power
USB C 16Pin x2 for two serial port
5.1kOhm Resistor x4
1kOhm Resistor x5
10kOhm Resistor Network x1 for Address pull up
Three Pin OnOff Switch x1 for Power Switch
6 Bit DIP switch x1 for programming the Serial Port IO address.
Three Pin OnOff Switch x1 for IEI switch
CH340C SMD Component X2 for USB TTL IC handle
74HCT688 x1 for Address Programme comparing
Z84C42BB6 x1 Dual Serial Z80 IO Chip



==CPU and Clock Module (8MHz)
10uF 10V+ Capacitor x1 for auto reset on power
104 Capacitor x3 for decoupling
LED x1 for Clock Signal Visualizing
10Kohm  resistor x5 for Signal Pull ups
THT 6mm Push Button x1 for Reset Signal
SPDT Switch x1 for Auto Clock Enable and Disable
Z84C0008P x1 for the main CPU of the Computer (8MHz Clock Zilog80)
74HCT04 x1 for clock buffering
7.3728MHz Crystal x1 for Clock signal generation


==Memory Module
LED x1 for Enabled Signal Visualization
10KOhm Resistor Network x4 for Pulling Up the 74HCT688
DIP switch 8 bit x4 for masking and setting Address Compare signal
SDPT Switch x1 for Enabling and Disabling the Memory for User
74HCT688 x2 for comparing the address bus with the user input
74LS32 x1 Or Gate for Signal Buffering

28C256 or 62256 x1 for Memory IC itself (EEPROM or RAM)
